# Footer Editing Guide

## Where to Edit Footer Items

Your CMS footer items can be edited in **two different places** depending on what type of footer content you want to edit:

---

## 1. Footer Menu Links (Navigation Links)

**Location:** `Appearance → Menus` in the admin panel

### How to Edit Footer Menu Items:

1. **Go to:** `Appearance → Menus` (in the left sidebar)
2. **Create or Select a Menu:**
   - If you don't have a menu yet, click "Create New Menu" and give it a name
   - Or select an existing menu from the dropdown

3. **Add Menu Items:**
   - Use the "Add Menu Items" panel on the right
   - Click quick-add buttons (Home, Pages, Posts, Categories, Shop, Blog, Quote, Custom Link)
   - Or use the dropdown to select specific items

4. **Assign to Footer:**
   - Scroll down to the "Menu Locations" section
   - Find "Footer Menu" 
   - Select your menu from the dropdown
   - The menu will automatically appear in the footer

5. **Edit Existing Items:**
   - Click "Edit" next to any menu item to change its label or URL
   - Drag items to reorder them
   - Drag items under others to create submenus
   - Click "Delete" to remove items

6. **Save:** Click "Save Menu" button

---

## 2. Footer Widgets (Content Blocks)

**Location:** `Appearance → Widgets` in the admin panel

### How to Edit Footer Widgets:

1. **Go to:** `Appearance → Widgets` (in the left sidebar)

2. **Select a Footer Widget Area:**
   - You'll see a list of widget areas
   - Click on one of these footer areas:
     - **Footer Column 1** (`footer-1`)
     - **Footer Column 2** (`footer-2`)
     - **Footer Column 3** (`footer-3`)
     - **Footer Column 4** (`footer-4`)

3. **Add Widgets:**
   - Click "Add New Widget" button
   - Fill in the form:
     - **Widget Title:** The heading for your widget
     - **Widget Type:** Choose from:
       - Text Widget (plain text)
       - Custom HTML (HTML code)
       - Recent Posts (shows latest blog posts)
       - Categories (shows post categories)
       - Pages (shows page links)
       - Search (search form)
       - RSS Feed (RSS feed items)
     - **Content:** Enter your content based on widget type
     - **Widget Order:** Number for ordering (lower = appears first)
     - **Active:** Check to make widget visible
   - Click "Save Widget"

4. **Edit Existing Widgets:**
   - Click "Edit" next to any widget
   - Modify the title, type, content, or order
   - Click "Save Widget"

5. **Delete Widgets:**
   - Click "Delete" next to any widget
   - Confirm deletion

6. **Reorder Widgets:**
   - Drag widgets up and down to change their order
   - Lower numbers appear first

---

## Default Footer Items

If you haven't added any widgets yet, the footer will show default items:
- **About** - Company name and tagline
- **Quick Links** - Footer menu items (if assigned)
- **Legal** - Privacy Policy and Terms links
- **Contact** - Link to quote page

Once you add widgets to any footer column, these default items will be replaced with your custom widgets.

---

## Quick Access Links

- **Edit Footer Menus:** Click `Appearance → Menus` in admin sidebar
- **Edit Footer Widgets:** Click `Appearance → Widgets` in admin sidebar
- **View Site:** Click "View Site" in the admin header to see your changes

---

## Tips

1. **Footer Menus:** Best for navigation links (About Us, Contact, Services, etc.)
2. **Footer Widgets:** Best for content blocks (About text, social links, recent posts, etc.)
3. **Multiple Columns:** You can add widgets to multiple footer columns (1-4) to create a multi-column layout
4. **Widget Order:** Use widget order numbers to control which widgets appear first in each column
5. **Active Status:** Uncheck "Active" to temporarily hide a widget without deleting it

---

## Need Help?

- Make sure you're logged in as an **Administrator**
- Widgets must be set to "Active" to appear on the frontend
- Menu must be assigned to "Footer Menu" location to appear in footer
- Changes save automatically - refresh your frontend to see updates
